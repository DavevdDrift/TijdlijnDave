History of Responsive Webdesign

3500 v.chr

Het Schrift

Een van de belangrijkste communicatiemiddelen in de geschiedenis van de mensheid is het schrift. Door middel van het schrift konden mensen snel over een groot gebied informatie overdragen. Hieruit ontstond ook het boek, waardoor waarin veel informatie kan worden opgeslagen. Hierdoor konden mensen makkelijk veel informatie overbrengen. Dit werd gedaan door middel van tekst, maar ook door afbeeldingen. Het maken van een boek duurde echter enorm lang en er was maar een klein procent van de bevolking die kon schrijven.

1450

De Drukpers en de Krant

Een lange tijd na de uitvinding van het schrift ontstond er opnieuw een revolutionaire uitvinding. Deze uitvinding is de drukpers. De drukpers is bedacht door Johannes Gutenberg. De drukpers heeft ervoor gezorgd dat de verspreiding van informatie sneller ging dan ooit. Dit komt omdat teksten nu sneller geproduceerd konden worden. Een krant is een voorbeeld van wat een drukpers voor invloed heeft gehad op de wereld. Nieuws kon nu razendsnel verspreid worden, terwijl dat zonder de drukpers nooit zo snel had kunnen plaatsvinden.

1822

De Computer

De computer is een van de meest revolutionaire apparaten die uitgevonden is. Vooral in de tweede wereldoorlog kreeg de ontwikkeling van computers een snelle vlucht. Hier ontstonden namelijk de eerste elektrische computers. De computer kon erg snel moeilijke berekeningen berekenen en het zorgde ook voor een snellere communicatie op grotere afstanden. De computer was het begin van een nieuw tijdperk. Er kon namelijk veel meer gedaan worden met computers en mensen vonden steeds meer dingen uit, die de pc weer telkens meer op de huidige pc’s lieten lijken.

1993

HTML

De computer gaf enorm veel mogelijkheden op allerlei gebieden. Een van de belangrijkste dingen die je met je computer kan doen is het browsen op het internet. Het internet bestaat uit pagina’s en ook dit is uitgevonden. Zo begonnen de eerste websites uit codes die we HTML noemen. Hiermee kon je dingen zoals de opbouw, de inhoud en dingen zoals links verwerken. Je kan er ook dingen mee aanduiden, bijvoorbeeld wat een paragraaf of een header is. Hierdoor was het ook voor het eerst mogelijk om jezelf goed te navigeren door het internet, omdat je met de links naar andere pagina’s zou kunnen komen.

1994

CSS

Met alleen HTML ziet een website er standaard uit en je kan de pagina er bijna niet mee opmaken. Vandaar dat CSS uitgevonden is. Met CSS konden website een eigen uiterlijk krijgen. Dingen zoals animaties, afbeeldingen en overige opmaak zijn door CSS mogelijk gemaakt. Hierdoor konden mensen hun website opmaken naar hun eigen wensen. Dit kan op enorm veel manieren gedaan worden en zelfs vandaag de dag is CSS het middel om de meest nieuwste design trends te verwerken in je website.

1995

JavaScript

Nadat HTML en CSS waren uitgevonden ontstond er ook het Javascript. Javascript gaf webdesigners de optie om hun website bijvoorbeeld gladder en meer reagerend te maken. Je kan met javascript ook veel extra functies maken, zoals het vergroten van een filmpje om het in fullscreen te zetten. Voor eigen applicaties is Javascript dus ook een goede manier om het te maken. Javascript kan het gebruikersgemak en ervaring dus vergroten.

2007

De IPhone van Apple

De volgende revolutionaire uitvinding die uitgevonden is, is de IPhone. Door de IPhone kon je ineens enorm veel functies met je telefoon doen. Je kon ermee bellen, spelletjes spelen, foto’s maken en nog vele ander functies. Een van de belangrijkste functies was dat je nu ook via je mobiel op het internet terecht kan komen. Het surfen op het internet vanuit je mobiel was echter niet goed geoptimaliseerd voor de mobiele gebruikers. Dit kwam omdat de ontwerpen van de websitepagina’s niet ontwerpen waren voor zulke kleine schermen.

2010

Concurrentie en Ontwikkeling

Voordat de IPhone was uitgebracht werden de meeste websites voor één soort grootte scherm ontworpen. Op een kleiner beeldscherm zou, zou de website dan niet mooi op je beeldscherm passen. Hierdoor moest je een hoop in- en uitzoomen, wat veel moeite kost. Dit heeft uiteindelijk geleid tot responsive webdesign. Dit houdt in dat de elementen op de website worden gerangschikt per verschillende soorten schermgrootten. Het is inmiddels de norm geworden om vanuit mobiele grootte te ontwerpen, en het ontwerp vanuit daar voor grotere beeldschermen te ontwerpen. Hierdoor is het ontwerp voor elk grootte scherm gebruikelijk.
